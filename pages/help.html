What is document.do?

Document.do let you read and create documents on Ethereum and Polygon.

Documents?

There are two types of documents:

  • Unnamed documents: published within a Ethereum or Polygon transaction.
  • Named documents: published on this Polygon contract.

The content of a document can be plain text, HTML or a router.

Drafts?

A draft is automaticly created when a named document isn't found. About drafts:

  • Encrypted in your browser with the name.
  • Anyone with the name can edit the draft. Don't use drafts to safe sensitive information.
  • Saved automaticly on centralized servers.

Named documents

Named documents are published in this contract. The contract is deployed on Polygon and unowned by anyone, so it is completely decentralized.

  • Only the owner of a named document can control it. No one else. Not document.do, no one. 100% censorship proof.
  • The owner can edit the contents of the document, change the owner or destroy the document.
