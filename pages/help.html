What is document.do?

Document.do let you read and publish decentralized documents with this Polygon contract.

Documents?

Documents are published in this contract. The contract is deployed on Polygon and unowned by anyone, so it is completely decentralized.

Documents can be:

  • Plain text
  • HTML documents
  • Unowned - anyone can edit the document.
  • Owned - Only the owner of the document can edit (or sell) it. No one else. Not document.do, no one. 100% censorship proof.
Drafts?

A draft is automaticly created when a document isn't published yet. About drafts:

  • Encrypted in your browser with the document name.
  • Anyone with the name can edit the draft. Don't use drafts to safe sensitive information.
  • Saved automaticly on centralized servers.

  • Owned documents - Owned documents can only be modified by the document owner. Others (including document.do) can't.
