“Named Document” - combination of values 'owner', 'name', 'content' added to the 'mapping' called 'names' in this smart contract.
“You” - buyer of a Named Document.
“Owner” - Polygon address you provide.
“We”, “Us”, or “Our” - document.do.

  • You agree with the User Agreement.
  • You aware of buying a Named Document means We add the values 'owner', 'name', 'content' to this smart contract.
  • You aware of the fact that only the Owner can modify the Named Document. We can't help you if you make mistakes/typos when providing the Owner. If you lose access to the Owner we can't help you.
  • We give no refunds.
  • If multiple parties buy the same Named Document at the same time the Owner of the first confirmed transaction will be added to the Named Document. The other parties will get their funds back. Transactions costs may apply.
